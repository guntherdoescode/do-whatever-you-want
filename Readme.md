This is the Do-Whatever-You-Want repo

Yes this is real, and no this isnt a joke.
As the title suggest you can do literally whatever you want in here.
Wanted to add a meme or a weird looking framework, go ahead!
Submit a PR, it automatically goes through the checks, and its in this repo!

A few rules tho:

1. Your code has to be compile at least and respect the rest of the project to an extent. If it crashes the entire project or your own module, it gets rejected.
2. No viruses or destructive code.
3. AI generated code might be tolerated.
4. any language is welcome.

Go nuts.

**Note:** To keep things somewhat organized, please put your code inside its own folder if necessary.
